# 路由 ```/api/data``` 

### ```POST /api/data/excellentUser```

获取优质接单方推荐接口(主页)

**请求参数**

| 参数名   | 类型           | 是否必填  | 描述       |
| ------   | ------------- | -------- | ---------- |
| page     | Int           | 否       | 页数(默认1) |
| tech     | Array<String> | 否       | 指定技术栈  |

| 请求头            | 类型    | 是否必填 | 描述     |
| ---------------- | ------  | -------- | -------- |
| Authorization    | Berear  | 否       | 身份验证密钥 | 
| Timestamp        | Long    | 否       | 10位秒级时间戳(UTC+8) | 

**响应结果**
```json
{
    "result": true,
    "message": "获取",
    "data": [
        {
            "id": "xxxxxx-xxxxx-xxxxx-xxxxx",
            "nickname": "张三",
            "avatar": "头像hash",
            "tech": ["Java", "Python", "C++"],
            "score": 80,
            "description": "软件开发工程师，擅长Java、Python和C++等多种编程语言。拥有丰富的项目经验，能够快速解决各种技术问题。熟悉软件开发流程，能够高效地完成项目任务。",
            "face_result": true,
            "create_time": 1700000000,
            "published_count": 10, //总派单数
            "contractor_count": 5, //总接单数
        },
    ]
}
```

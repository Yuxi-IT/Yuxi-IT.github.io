# 路由 ```/api/search``` 

### ```POST /api/search/user```

账户搜索接口

**请求参数**

| 参数名   | 类型    | 是否必填  | 描述     |
| ------   | ------ | -------- | -------- |
| userid   | string | 否       | 用户ID |
| nickname | string | 否       | 用户昵称 | 
| page     | int    | 否       | 当前页码(默认1) | 

**响应结果**
```json
{
    "result": true,
    "message": "搜索成功",
    "data": [
        {
            "uid": "唯一账户id",
            "avatar": "账户头像hash",
            "nickname": "默认用户昵称",
            "createTime": "注册时间(11位时间戳)",
            "creditScore": 60, //信用分
            "certification": 1, // 0=实人认证 1=学生认证 2=企业认证
            "status": 0 //0=正常 1=封禁 2=待审核
        }
    ]
}
```

### ```POST /api/search/order```

订单搜索接口

**请求参数**

| 参数名   | 类型    | 是否必填  | 描述     |
| ------   | ------ | -------- | -------- |
| orderid  | string | 否       | 订单ID(精确匹配) |
| keys     | string | 否       | 订单关键词(支持标题或描述模糊搜索) |

**响应结果**
```json
{
    "result": true,
    "message": "搜索成功",
    "data": [
        {
            "orderId": "订单ID",
            "title": "订单标题",
            "description": "订单需求摘要(可截断)",
            "images": ["订单图片hash"],
            "price": 2000,
            "createTime": 1734357600,
            "deadline": 1736949600,
            "status": 0,
            "publisher": "唯一账户id"
        }
    ]
}
```

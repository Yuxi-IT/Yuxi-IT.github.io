# 路由 ```/api/auth``` 

### ```POST /api/auth/login```

登录接口

**请求参数**

| 参数名   | 类型    | 是否必填  | 描述     |
| ------   | ------ | -------- | -------- |
| phone    | string | 是       | 手机号 |
| password | string | 是       | 哈希后的密码 | 

**响应结果**
```json
{
    "result": true,
    "message": "登录成功",
    "data": {
        "token": "长期token",
        "accessToken": "短期token",
        "user": {
            "uid": "唯一用户id",
            "avatar": "用户头像hash",
            "nickname": "默认用户昵称(系统分配)",
            "createTime": "注册时间(11位时间戳)",
            "creditScore": 60 //信用分
        }
    }
}
```

### ```POST /api/auth/register```

注册接口

**请求参数**

| 参数名   | 类型    | 是否必填  | 描述     |
| ------   | ------ | -------- | -------- |
| phone    | string | 是       | 手机号 |
| code     | string | 是       | 短信验证码 |
| email    | string | 否       | 邮箱地址 |
| password | string | 是       | 哈希后的密码 | 


**响应结果**
```json
{
    "result": true,
    "message": "注册成功",
    "data": {
        "token": "长期token",
        "accessToken": "短期token",
        "user": {
            "uid": "唯一用户id",
            "avatar": "默认用户头像hash",
            "nickname": "默认用户昵称(系统分配)",
            "createTime": "注册时间(11位时间戳)",
            "creditScore": 60 //信用分
        }
    }
}
```
